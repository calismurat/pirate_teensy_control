# pirate_teensy_control
A ROS wrapper for teensy32 including cmd_vel subscriber to MD25 publisher, IR, PIR topics etc.

## Requirements:  
-Git  
-Teensyduino  
-Atom  
-Atom PlatformIO-IDE

## Install:  
git clone https://github.com/calismurat/pirate_teensy_control.git .

## Start:  
atom .

## Upload:  
Alt+Ctrl+b  
Alt+Ctrl+u
